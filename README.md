# Face Detection and Recognition System
## 📌 Project Overview
This project is a Face Detection and Recognition System developed using Python and OpenCV. It can detect human faces in real-time through a webcam, register new users, train a recognition model, and identify known faces.
This project was developed as part of the Artificial Intelligence Internship Program at Syntecxhub.
## 🚀 Features
- Real-time Face Detection
- Face Registration and Dataset Creation
- Face Recognition using LBPH Algorithm
- Multiple Face Detection Support
- Webcam Integration
- User Identification and Label Display
## 🛠 Technologies Used

- Python
- OpenCV
- NumPy
- Pillow
- LBPH Face Recognizer
